Controlador ElNet XP

Controlador Elnet xp para bombas de drenaje y aguas residuales.

Un controlador de conmutación automática se utiliza para operar bombas de drenaje y aguas residuales y se encuentra en casi todos los edificios públicos hoy en día. El controlador se utiliza para controlar automáticamente el nivel de agua de lluvia o aguas residuales, incluido el cambio automático entre dos bombas en cada operación para equilibrar las horas de trabajo y la introducción de una bomba fuera de servicio como refuerzo cuando sea necesario.

Encontramos estos sistemas en fábricas, hoteles, hospitales, universidades, edificios de oficinas y cualquier otro edificio público donde se requiera un sistema confiable para operar bombas para drenar agua de lluvia o aguas residuales.

características principales

Operación amigable en el idioma hebreo usando una pantalla gráfica colorida.

Registro y visualización de documentación en tiempo de hasta 1000 alertas y eventos.

Ejecutar una bomba fuera de servicio para impulsar a un nivel alto.

Reposición automática para equilibrar la jornada laboral.

Prevención de operaciones en vacío resultantes de la vibración de los flotadores.

Cambiar el retraso de operación de una manera fácil y sencilla.

RS485 Comunicación de control de estructura Modbus (opción TCP/IP).

Actualización breve de la bomba después de un apagado prolongado del sistema.

Operar bombas en situaciones de falla de flotación o sin retorno.

Algoritmo para resolver problemas en situaciones inusuales de El recurrente.

Un comando para activar una bocina en caso de falla.

Cese del funcionamiento de la bocina de falla después de un período de tiempo variable.

El software de prueba de cableado incluye un registro en el informe de eventos de que se ha probado el cableado.

Dos salidas de contacto seco de 150ma para falla general y falla de surf.
