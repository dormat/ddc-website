Controlador SuperBrain

Controlador SuperBrain con biblioteca de software

SuperBrain es un controlador independiente especialmente diseñado (DDC).

Para control y medición de sistemas de aire acondicionado, ventilación y calefacción.

(HVAC) incluidos los sistemas de energía, iluminación y otros sistemas electromecánicos del edificio.

SuperBrain es un controlador óptimo para el control computarizado de sistemas de aire acondicionado, calefacción, ventilación y plomería en edificios públicos.

SuperBrain ofrece una variedad de software avanzado especialmente diseñado

para sistemas electromecánicos en hoteles, hospitales, universidades, edificios públicos e industriales.
