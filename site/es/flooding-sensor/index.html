Sensor de inundación

Sensor de inundación

El Sensor de Inundación FLs es un sensor electrónico diseñado para la detección de cualquier tipo de inundación o derrame de líquidos y la prevención de posibles daños causados por dicha inundación.

El sensor está especialmente diseñado para aplicaciones de detección de inundaciones en edificios comerciales y planeado específicamente para instalaciones debajo de pisos de salas de computadoras, canalones de drenaje, salas eléctricas, cuartos de almacenamiento, etc.
